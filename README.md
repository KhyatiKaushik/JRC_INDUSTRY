# JRC Industries Consultancy & Limited

A professional website for JRC Industries Consultancy & Limited, specializing in defense ammunition products and solutions.

## Overview

JRC Industries is a leading provider of specialized defense ammunition products including RDX, HMX, PETN, hand grenades, artillery shells, and custom defense solutions. This website showcases our products, services, and provides a platform for client consultation.

## Features

### 🎯 Core Features
- **Responsive Design**: Fully responsive website that works on all devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Product Showcase**: Detailed information about defense ammunition products
- **Service Portfolio**: Comprehensive consulting and support services
- **Contact System**: Professional contact form with validation
- **Performance Optimized**: Fast loading with lazy loading and optimized assets

### 🛡️ Products Featured
- **RDX** (Cyclotrimethylenetrinitramine) - High-performance military explosive
- **HMX** (Octogen) - Premium explosive material with exceptional performance
- **PETN** (Pentaerythritol tetranitrate) - High-velocity explosive compound
- **Hand Grenades** - Complete range of tactical grenades
- **Artillery Shells** - Precision-manufactured ammunition
- **Custom Solutions** - Tailored ammunition solutions

1. **Open the website**: Open `index.html` in a web browser

   ```bash

   python -m http.server 8000
   
   # Using Node.js (if installed)
   npx serve .
   ```

## Customization

### Colors
The website uses CSS custom properties for easy color customization. Main colors are defined in `:root`:
- `--primary-color`: Main brand color
- `--secondary-color`: Secondary brand color
- `--accent-color`: Accent/action color

### Content
- Update company information in `index.html`
- Modify product details in the products section
- Customize services in the services section
- Update contact information

### Styling
- Main styles are in `css/styles.css`
- Responsive breakpoints can be adjusted in media queries
- Animation timings and effects can be modified in CSS and JavaScript
